The schema for data solution automation is used in various ways by different projects. In many cases this is bespoke, integrated into company solutions and frameworks.

In other cases software products use the schema (or parts of it) to work towards an integrated way of sharing design metadata.

Known implementations:

- [Agnostic Data Labs](https://agnosticdatalabs.com) - implements this schema and provides a dedicated UI and out-of-the-box templates. ADL has been actively contributing to the schema's evolution.
- [Taxonomy of ETL Automation Metadata (TEAM)](https://github.com/data-solution-automation-engine/TEAM) - creating and managing source-target mappings.
- [Virtual Data Warehouse (VDW)](https://github.com/data-solution-automation-engine/VDW) - open-source code generator for data solutions.
